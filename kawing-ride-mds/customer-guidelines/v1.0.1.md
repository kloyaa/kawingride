# Customer Responsibilities and Safety Guidance

Last updated: February 17, 2026.

This archived version reflects the first customer-facing guidance published for Kawing Ride.

## Honest Requests

Customers should create requests with accurate trip details so riders can respond responsibly.

Do not create fake requests, omit key facts on purpose, or change the trip materially after negotiation without explaining it clearly.

## Fair Use of Rider Time

Customers should use negotiation in good faith.

* Compare responses honestly.
* Reply within a reasonable time where possible.
* Cancel promptly if you no longer need the ride.

## Privacy Awareness

Use the platform in a way that protects your own privacy and the privacy of others.

Avoid sharing unnecessary personal details before they are needed for the trip.

## Safety Awareness

Kawing Ride provides better structure and accountability than broad public posting, but customers should still use judgment.

If something feels deceptive or unsafe, pause the trip and report the issue through the available process.

## Reporting Problems

If a rider acts dishonestly, violates privacy, or fails to honor agreed terms, submit a truthful report with relevant details where possible.
